3 - Sejam Xi (i N) v.a's Bernoulli com p = 0.5.

a) Prove que $X = \sum_{i=1}^{\infty} 2^{-i} X_i$ tem distribuio uniforme em [0,1].

Para provar que a varivel (X) definida como:

$$ X = \sum_{i=1}^{\infty} \frac{X_i}{2^i}$$

onde $X_i$ so variveis aleatrias de Bernoulli com (p = 0.5), tem distribuio uniforme no intervalo [0, 1], podemos seguir o seguinte raciocnio:

  1. Cada $X_i$ uma varivel aleatria de Bernoulli com parmetro (p = 0.5), o que significa que $X_i$ pode assumir os valores 0 ou 1, com igual probabilidade.
  2. Portanto, cada termo na soma $X_i / 2^i$ assume um valor no conjunto {0, 1/2, 1/4, 1/8,...}. Esses valores so potncias de (1/2) e, portanto, podem ser vistos como uma representao binria do nmero no intervalo ([0, 1]).
  3. A soma desses termos resulta em uma soma infinita de potncias decrescentes de (1/2), que exatamente a representao de um nmero real no intervalo ([0, 1]) em base binria.
  4. Dado que cada $X_i$ independente das outras e segue uma distribuio de Bernoulli, e que estamos somando termos que representam a parte fracionria de um nmero em binrio, a soma (X) segue uma distribuio uniforme no intervalo [0, 1]).

Portanto, a varivel (X) tem, de fato, uma distribuio uniforme no intervalo ([0, 1]).

Para provar que $ X = \sum_{i=1}^{\infty} \frac{X_i}{2^i} $ tem distribuio uniforme em ([0,1]), podemos simular a gerao de $X$ e plotar um histograma das amostras para verificar se elas seguem uma distribuio uniforme. Este cdigo ir gerar (10,000) amostras de $X$ e, em seguida, plotar um histograma dessas amostras. Se o histograma se parecer com uma distribuio uniforme no intervalo ([0,1]), isso fornecer evidncias de que $X$ tem distribuio uniforme nesse intervalo.

# Agora, vamos mostrar esse resultado computacionalmente com python

import numpy as np
import matplotlib.pyplot as plt

# Definiremos o nmero de amostras (escolha arbitrria)
num_samples = 10000

# Agora geraremos amostras de X
X_samples = np.zeros(num_samples)
for i in range(num_samples):
    X = 0
    for j in range(1, 55):  # Somar 54 termos para garantir preciso suficiente
        X += np.random.randint(0, 2) / (2 ** j)
    X_samples[i] = X

# Plotaremos histogramas da amostra
plt.hist(X_samples, bins=50, density=True, alpha=0.6, color='b', label='Amostras de X')

# Plotar a funo de densidade de probabilidade (PDF) terica da distribuio uniforme
x = np.linspace(0, 1, 1000)
pdf_uniform = np.ones_like(x)  # A PDF da distribuio uniforme  1 em todo o intervalo [0, 1]
plt.plot(x, pdf_uniform, 'r-', lw=2, label='PDF terica da distribuio uniforme')

plt.title('Histograma de Amostras de X e PDF Terica da Distribuio Uniforme')
plt.xlabel('Valor de X')
plt.ylabel('Densidade de Probabilidade')
plt.legend()
plt.show()

import numpy as np
import matplotlib.pyplot as plt

# Funo para gerar amostras uniformemente distribudas
def generate_uniform(n):
    samples = np.zeros(n)
    for i in range(n):
        sample = 0.0
        for j in range(1, 55):  # Somar 54 termos para garantir preciso
            X_i = np.random.randint(0, 2)  # Gerar uma amostra de Bernoulli com p=0.5
            sample += X_i / 2**j
        samples[i] = sample
    return samples

# Amostrar 1000 variveis uniformes em [0, 1]
n = 1000

import numpy as np
# Nmero de amostras
num_samples = 10000

# Gerar amostras uniformemente distribudas
U1 = np.random.uniform(0, 1, num_samples)
U2 = np.random.uniform(0, 1, num_samples)

# Aplicar a transformao de Box-Muller
Z0 = np.sqrt(-2 * np.log(U1)) * np.cos(2 * np.pi * U2)
Z1 = np.sqrt(-2 * np.log(U1)) * np.sin(2 * np.pi * U2)

# Plotar o histograma das amostras Z0
plt.hist(Z0, bins=50, density=True, alpha=0.6, color='b', label='Amostras geradas (Z0)')

# Plotar a funo densidade de probabilidade terica da distribuio Normal(0,1)
x = np.linspace(-4, 4, 1000)
pdf_normal = (1/np.sqrt(2 * np.pi)) * np.exp(-0.5 * x**2)
plt.plot(x, pdf_normal, 'r-', lw=2, label='PDF terica')

plt.title('Histograma das amostras da distribuio Normal(0,1) usando Box-Muller')
plt.xlabel('Valor')
plt.ylabel('Densidade de probabilidade')

Este cdigo Python usa o mtodo de Box-Muller para gerar amostras de uma distribuio normal padro (mdia 0, desvio padro 1) a partir de amostras uniformemente distribudas. Aqui est uma explicao passo a passo do cdigo:

  • Definimos o nmero de amostras desejadas (num_samples) como 10000.

  • Geramos duas sequncias de amostras uniformemente distribudas no intervalo [0, 1] usando np.random.uniform(0, 1, num_samples). Essas amostras so armazenadas em U1 e U2.

  • Aplicamos o mtodo de transformao de Box-Muller para converter as amostras uniformemente distribudas em amostras de uma distribuio normal padro. Este mtodo usa as seguintes frmulas:

$$Z_0 = \sqrt{-2 \ln(U_1)} \cdot \cos(2 \pi U_2) $$

$$ Z_1 = \sqrt{-2 \ln(U_2)} \cdot \sin(2 \pi U_2)$$

onde $U_1$ e $U_2$ so as amostras uniformemente distribudas e $Z_0$ e $Z_1$ so as amostras resultantes da distribuio normal padro. Essas amostras so armazenadas em $Z_0$e $Z_1$ .

  • Plotamos o histograma das amostras $Z_0$ usando plt.hist(), com 50 bins, densidade normalizada e transparncia de 0.6. Isso representa a distribuio das amostras geradas.

  • Plotamos a funo densidade de probabilidade (PDF) terica da distribuio normal padro (0,1) usando a equao da PDF.

  • Adicionamos um ttulo ao grfico, rtulos dos eixos x e y, e uma legenda para identificar as amostras geradas e a PDF terica.

  • Exibimos o grfico usando plt.show().

O resultado um histograma das amostras geradas $Z_0$ sobreposto funo densidade de probabilidade terica da distribuio normal padro. Isso permite visualizar como as amostras se distribuem em relao distribuio terica.
